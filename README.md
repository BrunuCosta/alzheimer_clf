Esse projeto foi desenvolvido para a disciplina de Inteligência Artificial da Universidade Federal do ABC.

Buscamos aplicar diversos algoritmos de classificação na base de dados Alzheimer's Disease Dataset do Kaggle. (Disponível sob a licença CC BY 4.0, permitindo que qualquer pessoa utilize o conjunto de dados em qualquer formato.)
https://www.kaggle.com/datasets/rabieelkharoua/alzheimers-disease-dataset/code

Efetuamos a análise do banco de dados, inferindo quais eram os atributos mais relevantes para o cálculo de predição, usando técnicas como Random Forest, HeatMap e Correlação de Pearson. A partir disso, aplicamos os seguintes algoritmos de classificação: Dummy, K-Vizinho (KNN), Rede Neural (Multilayer Perceptron) e Árvore de Decisão.
